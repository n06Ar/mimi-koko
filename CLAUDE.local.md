# Project-specific Claude settings

## Git commit language
Use English for commit messages.

## Development workflow
- All development tasks must start from a GitHub Issue
- Before starting work, read the Issue thoroughly to understand the requirements
- During and after work, post progress notes and completed items as comments on the Issue
- When resuming work, always read the Issue comments first to understand the current state
- `current-task.md` is not used in this project — GitHub Issues serve as the source of truth for task tracking
