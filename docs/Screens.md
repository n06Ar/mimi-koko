# 画面一覧

## 画面構成

```
認証系
├── ログイン画面
├── 新規登録画面
└── 家族グループ作成・招待画面

メイン
├── ダッシュボード
├── 支出一覧画面
├── 支出登録画面
├── レシートOCR画面
├── CSVインポート画面
├── 精算一覧画面
├── 精算詳細画面
├── 口座一覧画面
└── 設定画面
```

## 画面詳細

### ログイン画面
- パス: `/login`
- 要素: メールアドレス・パスワード入力、ログインボタン、新規登録リンク

### 新規登録画面
- パス: `/register`
- 要素: 名前・メールアドレス・パスワード・確認入力、アカウント作成ボタン

### 家族グループ作成・招待画面
- パス: `/group-setup`
- 状態1: グループ名入力 → グループ作成ボタン / 招待コード入力 → 参加ボタン
- 状態2: 招待コード表示（コピー・シェアボタン）→ はじめるボタン

### ダッシュボード
- パス: `/`
- 要素:
	- 月次サマリーカード（残高・収入・支出）
	- カテゴリ別支出
	- 最近の支出5件
	- 精算待ち通知カード
	- FAB（支出登録）

### 支出一覧画面
- パス: `/expenses`
- 要素:
	- 月切り替え・月合計
	- 日付別グループ表示
	- 非公開支出は🔒アイコン
	- 仮記録は「仮」バッジ
	- FAB（支出登録）

### 支出登録画面
- パス: `/expenses/new`
- 要素:
	- 金額入力（大きく表示）
	- 店名・メモ入力
	- カテゴリ選択（自動分類バッジ）
	- 日付選択
	- 口座・カード選択
	- 非公開トグル
	- 精算対象メンバー選択（公開時のみ）
	- レシート撮影ボタン

### レシートOCR画面
- パス: `/expenses/ocr`
- 状態1: カメラ撮影画面
- 状態2: 読み取り結果確認（店名・金額・日付）
- 注意: 画像はOCR処理後に破棄

### CSVインポート画面
- パス: `/settings/csv-import`
- 対応フォーマット: 楽天カード・三井住友カード・イオン銀行・楽天銀行・PayPay銀行・その他
- 状態1: 金融機関選択 → ファイル選択 → インポート開始
- 状態2: 結果表示（取込件数・マッチング済み・未マッチ）

### 精算一覧画面
- パス: `/settlements`
- 要素:
	- 未精算セクション（送金指示・精算済みにするボタン）
	- 精算済みセクション（履歴・済バッジ）

### 精算詳細画面
- パス: `/settlements/[id]`
- 要素:
	- 精算期間・合計金額
	- 送金指示（誰→誰・金額）
	- 対象支出一覧

### 口座一覧画面
- パス: `/settings/accounts`
- 要素: 種類別グループ（クレカ・銀行・電子マネー・現金）、非公開アイコン

### 設定画面
- パス: `/settings`
- 要素:
	- プロフィール（編集リンク）
	- 家族メンバー一覧・招待コード共有
	- 口座・カード管理リンク
	- 通知設定リンク
	- CSVインポートリンク
	- ログアウト
